# 🔌 Step 3: Integration Installation in Home Assistant

To view the data (including temperatures, hardware sensors, disks, PBS, VMs, and CTs), we will use the **Proxmox Extended Sensors** integration.

[Visual Installation Guide](#-Visual-Installation-Guide)

---

## 1. Installation via HACS

Being a custom integration, we must first add it to HACS:

1. Go to **HACS → Integrations**  
2. Click the **three dots** (top right)  
3. Select **Custom repositories**  
4. Add this repository: `https://github.com/Javisen/proxmox_sensors/`
5. In **Category**, select `Integration`  
6. Install it and **restart Home Assistant**

---

## 2. Integration Configuration

Once restarted:

1. Go to **Settings → Devices & Services**  
2. Click **Add Integration**  
3. Search for **Proxmox Extended Sensors**

---

## 3. Connection Data

The form is simple, but there are important details:

### 🔹 Host
- **On local network:** just the IP → `192.168.1.50`  
*(Do not add port or http/https)*  
- **From outside:** your domain → `proxmox.mydomain.com`  
*(The integration automatically detects http/https)*

### 🔹 Server type
- **PVE** → Proxmox Virtual Environment  
- **PBS** → Proxmox Backup Server  

### 🔹 Authentication method
- **Traditional login** (only PVE)  
- **API Token** (mandatory for PBS)

---

## 🔐 Option A: Login with User (without Token)

Only valid for **PVE**.

Fields:

- **User:** `user@realm`  
Examples:  
- `homeassistant@pve`  
- `root@pam`  
- **Password:** the user's password  
- **Node Name:** node name (as it appears in Proxmox)

---

## 🔐 Option B: Login with Token (recommended and mandatory for PBS)

Fields:

- **User:** `user@realm`  
- **token_id:** only the token name → `ha-token`  
*(Do not put `user@pve!token`)*  
- **Token_secret:** the Secret generated by Proxmox  

---

## ✅ Entity Selection (only in PVE)

After connecting, the integration will scan your server and you can choose what to monitor:

- **VMs**  
- **CTs**  
- **Physical disks**  
- **Storages**  

> [!TIP]  
> Select only what you need to keep Home Assistant clean and fast.

## ⚠️ Important note for PBS in shared environments (Tuxis, Hetzner, etc.)

If you use a **managed** or **multi‑tenant** PBS, such as Tuxis Free PBS:

- You will not see hardware sensors  
- You will not see temperatures  
- You will not see physical disks  
- You will not see node metrics  

This is normal because:

- You do not have access to the actual hardware  
- The provider hides the infrastructure  
- You do not have root permissions  
- You cannot access the real filesystem  

**Result:**  
The integration will only show empty sensors or no data.  
In future versions we will try to display custom datastore metrics.
