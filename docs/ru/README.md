# 🚀 Proxmox Extended Sensors

**Самая полная, эффективная и организованная интеграция для мониторинга Proxmox VE и PBS из Home Assistant.**

Эта интеграция разработана для продвинутых пользователей, которым необходим полный контроль над оборудованием без перегрузки сервера.  
В отличие от других решений, Proxmox Sensors Extended фокусируется на энергоэффективности, безопасной аутентификации через токены и безупречной визуальной организации.

---

## 🔥 Основные характеристики

### 🌡️ Расширенный мониторинг оборудования

**Не ограничивайтесь только использованием процессора. Посмотрите, что на самом деле происходит «под капотом»:**

* **Температура в реальном времени:** Ядра процессора, VRM и накопители NVMe/SSD/HDD.
* **Механические датчики:** Скорость вращения вентиляторов (RPM) и напряжения материнской платы.
* **Умные датчики:** Создаются только те объекты, которые передают валидные данные, что сохраняет чистоту вашей системы.

**(Примечание: Требуется установка lm-sensors на хосте Proxmox).**

---

### 🧠 Оптимизация производительности

**Разработано с учетом оборудования с ограниченными ресурсами:**

* **DataUpdateCoordinator:** Минимизирует количество вызовов API Proxmox, чтобы избежать перегрузки процессора сервера.
* **Silent SSL:** Автоматическая проверка SSL-сертификатов (включая самоподписанные) без заполнения журналов ошибками.

---

### 🗄️ Продвинутый Proxmox Backup Server (PBS)

* **Внешний режим:** Легко подключайтесь к удаленным серверам PBS, используя только домен.
* **Мониторинг задач:** Детальный статус последнего бэкапа, работы Garbage Collector или задач проверки (Verify).

---

### 🎨 Динамичный и организованный интерфейс

* **Smart Dashboard:** Датчики автоматически группируются по устройствам:  
  1. Узел (Node)  
  2. Физические диски  
  3. Виртуальные машины  
  4. Контейнеры  
  5. Хранилища (Storages)
* **Auto-Naming:** Автоматические префиксы (например, `pv1-cpu-temp`) для поддержания логического порядка в ваших панелях управления.

## Основные датчики

## PVE

### 🖥️ Датчики оборудования (PVE и PBS)

Температуры процессора • Температуры VRM • Температуры NVMe/SSD/HDD  
Скорость вентиляторов (RPM) • Напряжения • Датчики энергопотребления • Объекты `pvesensors`  
• Температура чипсета

---

### 🧠 Состояние узла

Использование CPU (%) • Использование RAM (%) • Занято/Всего RAM  
Время аптайма (uptime) • Load average • CPU I/O Wait

---

### 💾 Диски

Общая емкость • Использованное место (ГБ и %)  
Уровень износа (NVMe) • Статус SMART (если доступен)

---

### 🖥️ Виртуальные машины (QEMU)

Использование CPU (%) • Использование RAM (%) • Сетевой трафик Tx/Rx  
Статус (включена/выключена) • Автоматический или ручной выбор

---

### 📦 Контейнеры (LXC)

Использование CPU (%) • Использование RAM (%) • Сетевой трафик Tx/Rx  
Статус • Автоматический или ручной выбор • И многое другое

---

### 🗄️ Proxmox Backup Server (PBS)

Использование хранилища (ГБ и %) • Количество бэкапов  
Статус Garbage Collector • Статус последней задачи бэкапа  
• Полная информация о задачах и другое
